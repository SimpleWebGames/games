Memory
Règles du jeu :
Commencez par cliquer sur deux cartes selon votre choix
  • 1. Si vous trouvez la paire, alors les deux cartes disparaîtront

  • 2. Si les deux cartes ne sont pas identiques, alors vous aurez quelques secondes pour essayer de mémoriser l'emplacement des cartes et celles-ci se retourneront face cachées
But du jeu :
Trouvez toutes les paires en un minimum de temps !


Commencez à jouer !
